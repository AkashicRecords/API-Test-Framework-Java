# API Test Framework

## Introduction
This is a simple API test framework using Java, Maven, and TestNG.

## Setup
- Install Java and Maven.
- Clone the repository.
- Navigate to the project directory and run `mvn test`.

## Libraries
- RestAssured 5.2.0
- Gson 2.9.0
- TestNG 7.6.1
